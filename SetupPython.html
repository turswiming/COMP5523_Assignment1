1 Official Python Tutorial

This course assumes that we have learned Python before. If you are not familiar with Python, I recommend reading the official Python Tutorial, which is considered one of the best resources for learning Python.

Get help information in Python: we may forget the usage of a specific function sometimes. In such cases, we can utilize the built-in help function to access detailed information about the particular function. For example, if we forget the usage of the print function, we can obtain help by the following command:

help(print)

Running this command will display the following information, explaining the meaning of each argument:

Help on built-in function print in module builtins:

print(...)
    print(value, ..., sep=' ', end='\n', file=sys.stdout, flush=False)

    Prints the values to a stream, or to sys.stdout by default.
    Optional keyword arguments:
    file:  a file-like object (stream); defaults to the current sys.stdout.

    sep:   string inserted between values, default a space.
    end:   string appended after the last value, default a newline.
    flush: whether to forcibly flush the stream.

2 Setup Python Environment

To complete the programming assignments for this course, we need Python3 and several Python packages, including Numpy, Pillow, and PyTorch. If you are already familiar with setting up and managing your Python environment, feel free to do it in your preferred manner. Alternatively, if you need guidance, you can follow the step-by-step instructions outlined in this section to set up your Python 3 development environment.

2.1 Install Anaconda

The most convenient approach to set up a Python environment is by installing Anaconda, a Python environment management system that includes numerous useful third-party packages. Anaconda already comes pre-installed with almost all the packages we require.

To get started, please visit the Anaconda download page and select an appropriate version of Anaconda3 for your system. Depending on your preference, you can choose either the graphical installer or the command line installer. If needed, refer to the Installation page for further instructions.

2.2 Update Environment Variables

In order to run and debug your Python code on a terminal, you need to execute the following scripts before using Python. Additionally, you have the option to add these scripts to ~/.bashrc or ~/.zshrc to ensure that they run automatically every time you start your computer.

export ANACONDA_ROOT=your_own_anaconda3_path
export PATH=$ANACONDA_ROOT/bin:$PATH
export PYTHONPATH=$ANACONDA_ROOT/lib/python3.8/site-packages:$PYTHONPATH

Now you can start your programming:

# run
python code/filter.py

# debug
python -m pdb code/filter.py

2.3 Install PyCharm

To run your Python code, you can also utilize the PyCharm integrated development environment (IDE), which offers a variety of convenient tools for managing, running, debugging, and deploying Python programs. If you are interested, you can download and install PyCharm by visiting this [link] https://www.jetbrains.com/pycharm/download/) for the community version.

Once you have installed PyCharm, you will need to configure it to use the Anaconda distribution that you have previously installed. Here are the steps:

  1. Open PyCharm Preference by pressing -PyCharm -> Preferences
  2. Search “Python interpreter” in the top left corner.
  3. Click on “Add interpreter-> Add Local interpreter” at the right side.
  4. In the new window that appears, choose “Conda Environment”.
  5. Select “Interpreter” and “Conda executable” as the one you installed:
    • Example on Win10:
      • Interpreter: C:\Users\<USERNAME>\anaconda3\python.exe
      • Conda executable: C:\Users\<USERNAME>\anaconda3\Scripts\conda.exe
    • Example on macOS:
      • Interpreter: /Users/<USERNAME>/anaconda3/bin/python
      • Conda executable: no need to select.
  6. Check “Make available to all projects” and click “OK”.
  7. Now come back to previous interface. Wait and then click “OK” again. –>

After the configuation, PyCharm projects can use Anaconda as the default Python environment.
